# Предупреждение о возрастном ограничении

Если на вашем сайте присутствует контент с возрастным ограничением, включите уведомление об этом при первом посещении сайта. Для этого:

* В разделе настроек **«Правовые настройки» → «Уведомления посетителей сайта»** добавьте предупреждение о контенте 18+.

<figure><img src="../.gitbook/assets/legal 18+ ru.png" alt="" width="563"><figcaption></figcaption></figure>

* Напишите текст уведомления и текст для кнопки входа на сайт и выхода.

<figure><img src="../.gitbook/assets/legal 18+ text ru.png" alt="" width="563"><figcaption></figcaption></figure>

{% hint style="info" %}
При нажатии кнопки выхода с сайта посетитель будет перенаправлен на сайт google.com.
{% endhint %}

<figure><img src="../.gitbook/assets/18+ example ru.png" alt=""><figcaption></figcaption></figure>

### Как сделать предупреждение 18+ на отдельной странице <a href="#one-page" id="one-page"></a>

* Добавьте на сайт составную страницу.

<figure><img src="../.gitbook/assets/add custom page ru.png" alt="" width="323"><figcaption></figcaption></figure>

* Дайте ей имя (например, «Ню-съемки»). Сделайте страницу видимой на сайте.

<figure><img src="../.gitbook/assets/18+ one page 1 ru.png" alt="" width="563"><figcaption></figcaption></figure>

* Выберите вариант **«Составить самостоятельно»** и добавьте на страницу текстовый блок. Напишите в нем текст предупреждения.
* Нажмите на <mark style="background-color:blue;">**(+)**</mark> и перейдите во вкладку **«Кнопки»**. Выберите блок с двумя кнопками.

<figure><img src="../.gitbook/assets/add block template ru.png" alt=""><figcaption></figcaption></figure>

* Перейдите в настройки кнопки, дайте им название и выберите действие **«Открыть страницу сайта»**:
  * Первая кнопка подтвердит, что посетителю есть 18, и перенаправит на страницу 18+.
  * Вторая кнопка перенаправит на другую страницу сайта. Например, на главную.

<figure><img src="../.gitbook/assets/18+ button actions ru.png" alt=""><figcaption></figcaption></figure>

Пример страницы с предупреждением 18+:

<figure><img src="../.gitbook/assets/example custom page 18+ warning ru.png" alt=""><figcaption></figcaption></figure>
