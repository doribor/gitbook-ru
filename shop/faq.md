# Частые вопросы по магазину

* **Как оформить витрину на сайте?**\
  Нужно создать отдельную страницу для продажи. Посмотрите инструкцию: [products.md](site/products.md "mention").
* **А есть отдельный раздел «Витрина» для сайта?**\
  Пока что нет. В будущих обновлениях появится блок с витриной — он будет получать информацию о товарах автоматически. Такой блок можно будет встроить на любую страницу сайта.
* **Как оформить витрину на диске?**\
  В галерею можно добавить до 4-х товаров. Они будут отображаться в каждой папке под фотографиями в галерее. Смотрите инструкцию: [products.md](gallery/products.md "mention").
* **Есть корзина для оплаты нескольких товаров?**\
  Услуги, связанные с фотографией, чаще всего подразумевают покупку одной услуги клиентом за раз. Пока что корзины для отдельных товаров нет. Она может появиться в будущих обновлениях платформы.\
  Корзина есть в галереях на диске — при [продаже цифровых фотографий](gallery/digital-photos.md).
* **Как добавить микроразметку (Schema.org, Open Graph и др.) для магазина?**\
  Пока что такой возможности нет. В дальнейших обновлениях мы добавим блок «Витрина товаров» и микроразметку для поисковых систем в этом блоке. Вручную ничего добавлять не придется.
