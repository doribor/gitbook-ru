# Настройка доступа к странице после оплаты

Вы можете создавать на сайте страницы с ограниченным доступом, доступные только для определенного круга пользователей (например, как личный кабинет). Список пользователей, которым будет открыт доступ, и срок доступа к странице вы настраиваете самостоятельно.

Для доступа к странице пользователи получают уникальный одноразовый пароль.

### Как работает одноразовый пароль?

* При входе на закрытую страницу пользователю нужно ввести почту для доступа.
* На эту почту придет письмо с одноразовым паролем.
* Пароль отправляется только на ту почту, которая есть в базе клиентов для данной страницы.

<figure><img src="../../.gitbook/assets/otp page example ru.png" alt=""><figcaption></figcaption></figure>

### Как добавить страницу с доступом по одноразовому паролю

* Одноразовый пароль устанавливается для всех страниц, кроме страницы-заставки. Перейдите в настройки страницы во вкладку **«Доступ»** и выберите тип доступа **«Одноразовый пароль»**.

<figure><img src="../../.gitbook/assets/customer access password ru.png" alt="" width="563"><figcaption></figcaption></figure>

После у вас появится вкладка **«Клиенты»** с базой клиентов, которым предоставлен доступ к странице. Здесь вы можете добавлять новых клиентов и редактировать существующих.

<figure><img src="../../.gitbook/assets/add customer ru.png" alt=""><figcaption></figcaption></figure>

В настройках клиента можно изменить имя, почту клиента, а также ограничить срок доступа к странице.

<figure><img src="../../.gitbook/assets/customer edit ru.png" alt="" width="563"><figcaption></figcaption></figure>

### Как настроить доступ в личный кабинет после оплаты

* [Создайте товар.](../general/add-product.md) После этого настройте переход на закрытую страницу.
* В карточке товара во вкладке **«Действия после заказа»** включите опцию **«Перейти на страницу»** и выберите **«На страницу сайта»**.
* Выберите страницу для перехода и срок доступа, если хотите его ограничить. Нажмите на кнопку **«Добавить»**.

<figure><img src="../../.gitbook/assets/add product password page ru.png" alt="" width="563"><figcaption></figcaption></figure>

{% hint style="info" %}
После покупки товара почта покупателя добавляется в базу клиентов **автоматически**, если в карточке товара выбран переход на страницу с одноразовым паролем.
{% endhint %}

* Вернитесь на страницу с товаром и выберите кнопку, по которой будет совершена покупка. В настройках кнопки выберите действие **«Открыть товар/услугу»** и добавьте карточку товара из списка.

<figure><img src="../../.gitbook/assets/add product card site ru.png" alt="" width="563"><figcaption></figcaption></figure>

{% hint style="success" %}
После оплаты покупатель будет перенаправлен на страницу с закрытым доступом.
{% endhint %}



На закрытую страницу можно добавить информацию, доступную только для покупателей, например, [видеоуроки](../../site/content/protect-videos.md) или [файлы для скачивания](../../site/content/add-file.md).
