# Продажа сертификатов на сайте

Вы можете оформить красивую страницу и принимать оплату за сертификаты прямо на сайте. Для приема онлайн-платежей подключите [платежную систему](../payment/payment-systems.md).

{% hint style="success" %}
Создайте свой дизайн сертификата или скачайте один из [**готовых шаблонов**](https://free.wfolio.pro/disk/shablony-sertifikatov-74mzdg), которые подготовили наши дизайнеры.
{% endhint %}

### Как добавить сертификат в магазин

* Перейдите в раздел **«Магазин»** и нажмите **«Добавить товар»**.
* В карточке укажите название и описание сертификата; стоимость. Загрузите изображение — оно будет отображаться перед оплатой.

Если оформляете сертификат вручную, настройте [сбор контактных данных](../general/add-product.md#form) (например, email, номер телефона) через форму заказа. А если сертификат одинаковый для всех клиентов, можно настроить автоматическое скачивание после оплаты.

#### Автоматическое скачивание сертификата

* Создайте проект на облачном диске и загрузите туда сертификаты.

{% hint style="info" %}
Если сертификаты индивидуальные — этот способ **не подойдет**. В этом случае используйте только сбор контактных данных.
{% endhint %}

* Скопируйте ссылку на сертификат.

<figure><img src="../../.gitbook/assets/copy certificate link ru.png" alt=""><figcaption></figcaption></figure>

* Вернитесь в **«Магазин»** и настройте действие после заказа:
  * Выберите **«Перейти на страницу» → «Внешний адрес»**.
  * Вставьте ссылку на сертификат.

Готово! Сертификат скачается сразу после оплаты. Теперь добавим сертификат на страницу сайта.

### Как создать страницу для продажи

* В разделе **«Сайт»** нажмите на кнопку **«‎Добавить на сайт»** и выберите [составную страницу](../../site/pages/custom-page.md).

<figure><img src="../../.gitbook/assets/add custom page ru.png" alt="" width="323"><figcaption></figcaption></figure>

* После создания страницы нажмите **«‎Составить самостоятельно»**.
* Нажмите на кнопку <mark style="background-color:blue;">**(+)**</mark> для добавления блоков. Откройте вкладку **«Кнопки»** и выберите подходящий вариант блока с кнопкой.
* Добавьте на страницу информацию о сертификате, изображения и текст.
* После заполнения страницы перейдите в настройки кнопки.

<figure><img src="../../.gitbook/assets/sell certificate.png" alt=""><figcaption></figcaption></figure>

* Выберите действие **«Открыть товар/услугу»** и добавьте карточку сертификата из списка. **Сохраните изменения.**

<figure><img src="../../.gitbook/assets/add certificate to page ru.png" alt="" width="563"><figcaption></figcaption></figure>
