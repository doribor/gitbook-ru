# Прием оплаты онлайн

С помощью сайта вы можете продавать услуги, цифровые и физические товары с оплатой онлайн. В галереях — цифровые и печатные фотографии, фотокниги и другие товары. В этой статье мы расскажем, какие шаги нужно выполнить, чтобы успешно подключить онлайн-оплату и настроить продажи.

Платежная система подключается для сайта, который полностью готов к работе и открыт для посетителей. Перед подключением специалисты платежной системы проверят страницу, на которой вы планируете продавать товары. Важно, чтобы страница была доступна для просмотра и содержала доступные для взаимодействия товары. Убедитесь, что на странице есть вся необходимая информация.&#x20;

Если вы используете только диск, для проверки укажите прямую ссылку на галерею с витриной товаров — каждая галерея на диске доступна по индивидуальной ссылке.

Перед подключением платежной системы проверьте ваш сайт или галерею на диске по чек-листу ниже.



{% stepper %}
{% step %}
### Документы

* Подготовьте и добавьте **оферту**. \
  Это необходимый документ для подключения платежной системы, который закрепляет условия сделки.&#x20;
* Добавьте **политику конфиденциальности**.\
  Этот документ понадобится, так как при оплате покупатели передают вам свои данные.
* Убедитесь, что ссылки на оферту и политику конфиденциальности отображаются внизу страницы.\
  \
  Мы подготовили шаблоны документов и пошаговые инструкции о том, как добавить их на сайт и в галереи на диске:\
  [Как добавить оферту](../../legal/terms.md)\
  [Как добавить политику конфиденциальности](../../legal/privacy-policy.md)
{% endstep %}

{% step %}
### Контакты и реквизиты

В оферте укажите:

* Ваши ФИО и реквизиты самозанятого или ИП.
* Телефон и адрес электронной почты.

Ваши контакты также можно указать в визитке — убедитесь, что она включена и отображается на сайте/в галерее. [Визитка и контакты](../../contacts.md)
{% endstep %}

{% step %}
### Страница с товарами

* Создайте товары с оплатой наличными в разделе **«Магазин».**
* Добавьте описание товаров – убедитесь, что указаны:
  * название
  * цена&#x20;
  * подробная информация о товаре/услуге
  * информация о том, как покупатель получит товар/услугу после оплаты.
* Настройте действие после заказа.
* Оформите страницу с товарами на сайте. [Как разместить товары на сайте](../site/products.md)
* Убедитесь, что сайт открыт для посетителей. \
  Открыть сайт для посетителей можно в разделе настроек **«Настройки сайта» → «Режим разработки сайта»**.
* **Если у вас только диск**, достаточно оформить одну галерею для продажи: загрузите фотографии и добавьте товары с оплатой наличными. [Как разместить товары в галерее](../gallery/products.md)

{% hint style="warning" %}
Если в настройках товара выбрать способ оплаты «Онлайн-оплата», до подключения платежной системы кнопки для перехода к товарам на сайте и в галерее будут неактивны. Для успешного прохождения проверки установите способ «Оплата наличными» — так кнопки будут кликабельными, а весь сценарий покупки доступен для проверки.\
\
Если вы планируете продавать цифровые фотографии на диске, для проверки добавьте витрину товаров в галерею. Продажа цифровых фотографий будет работать только после успешного подключения платежной системы.
{% endhint %}
{% endstep %}

{% step %}
### Подключение платежной системы

* Выберите удобную платежную систему из доступных.
* Зарегистрируйтесь на сайте платежной системы и пройдите проверку.
* После одобрения заявки, настройте платежную систему.
* Проведите тестовый платеж.
* Если тестовый платеж прошел успешно, вы можете принимать реальные платежи.

[Список платежных систем с инструкциями по подключению](payment-systems.md)
{% endstep %}

{% endhint %}
