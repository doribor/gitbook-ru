# Прием оплаты онлайн

С помощью сайта вы можете продавать услуги, цифровые и физические товары с оплатой онлайн. В галереях — цифровые и печатные фотографии, фотокниги и другие товары.&#x20;

Перед подключением онлайн-оплаты специалисты платежной системы проверят ваш сайт или галерею на диске. Мы подготовили чек-лист: когда все пункты выполнены, можно переходить к подключению платежной системы.

{% hint style="success" %}
Для проверки вы можете указать любую страницу с товарами. Платежная система подключается к вашему аккаунту на Wfolio — после подключения принимать оплату онлайн вы сможете как на сайте, так и на диске.
{% endhint %}

### Страница с товарами

* На сайт или в галерею добавлены товары/услуг&#x438;**.**
* На странице товара указаны:
  * название
  * цена&#x20;
  * подробная информация о товаре/услуге
  * информация о том, как покупатель получит товар/услугу после оплаты.
* Настроено действие после заказа: письмо после оплаты, переход на страницу.
* Для товара выбрана оплата наличными.
* **Если у вас тариф с сайтом:** сайт полностью оформлен, открыт для посетителей — по адресу сайта открывается главная страница сайта, а не визитка. [Режим разработки сайта](../../site/settings/development-mode.md)
* **Если у вас только диск:** галерея с товарами готова для просмотра, в ней есть открытые папки.&#x20;

#### Подключение платежной системы.

* Выберите удобную платежную систему из списка доступных.
* Зарегистрируйтесь на сайте платежной системы и пройдите проверку.
* После одобрения заявки, настройте платежную систему.
* Проведите тестовый платеж.
* Если тестовый платеж прошел успешно, вы можете принимать реальные платежи.

[Список платежных систем с инструкциями по подключению](payment-systems.md)
