# Продажа цифровых фотографий

Вы можете использовать облачный диск Wfolio для онлайн-продажи цифровых фотографий.

### Как это работает

* Для любого проекта можно включить режим продажи фотографий.
* После загрузки фотографий на них автоматически наложится водяной знак.
* Клиент добавляет снимки в корзину. На почту направляется письмо о создании корзины — к ней можно вернуться по ссылке в письме.
* После оплаты онлайн клиент скачивает фотографии без водяного знака. Это можно сделать двумя способами:
  * Скачать из корзины сразу после оплаты.
  * Перейти в корзину по ссылке в письме — после оплаты оно направляется клиенту на почту.

{% hint style="success" %}
Для продажи фотографий подключите [платежную систему](../payment/payment-systems.md).
{% endhint %}

### Пример оплаченной корзины

По кнопке «Скачать все файлы» на устройство сохранится архив с фотографиями — для просмотра его нужно распаковать. Купленные фотографии можно скачать по отдельности — по кнопке скачивания рядом с фотографией.

### Как включить продажу фотографий?

* [**Подключите платежную систему**](../payment/payment-systems.md) для приема оплаты.
* В основных настройках проекта выберите тип проекта **«Продажа фотографий»**.

<figure><img src="../../.gitbook/assets/set up selling photos ru.png" alt="" width="563"><figcaption></figcaption></figure>

* Укажите цену за одну фотографию. Также можно настроить до **трех скидок** (в процентах), которые зависят от количества выбранных фотографий в корзине.

<figure><img src="../../.gitbook/assets/photo price ru.png" alt="" width="563"><figcaption></figcaption></figure>

#### Раздел «Корзины»

В проекте с продажей фотографий отображается раздел **«Корзины»**. В нем можно посмотреть, какие фотографии отобрали покупатели. Информация о корзинах в проекте отображается для всех заказов — оплаченных или в процессе.

<figure><img src="../../.gitbook/assets/project carts ru.png" alt=""><figcaption></figcaption></figure>

#### Имя клиента в форме заказа <a href="#client-name" id="client-name"></a>

Имя клиента предлагается ввести всегда — это позволяет идентифицировать корзину. Перейдите в **«Настройки диска»** и выберите, как отображается запрос имени клиента при продаже фотографий и [создании избранного](../../gallery/selections/make-favorites-list.md).

Полезно, если вам нужно запрашивать **только имя** или сначала **фамилию, а потом имя**.

<figure><img src="../../.gitbook/assets/client name drive ru.png" alt="" width="563"><figcaption></figcaption></figure>

{% hint style="info" %}
#### Тестовая покупка фотографий

Проверьте продажу фотографий после подключения платежной системы.

После покупки заказ фотографий должен появиться в разделе «Магазин».

Если заказ после оплаты не появился в разделе **«Заказы»** или попал в **«Незавершенные»** спустя несколько часов, исправьте настройки HTTP-уведомлений на сайте [платежной системы](../payment/payment-systems.md).
{% endhint %}

### Частые вопросы

* **Клиент оплатил фото, но не может их скачать. Почему?**\
  Убедитесь, что в вашей платежной системе настроены HTTP-уведомления. Если они настроены неправильно, Wfolio не получит уведомление об успешной оплате заказа, а фото не получится скачать.\
  Подробности настроек есть в инструкциях для каждой платежной системы: [ЮKassa](../../payments/yookassa.md#http), [Robokassa](../../payments/robokassa.md#podklyuchit-robokassa), [Т-Банк](../../payments/tbank.md#podklyuchenie-t-banka).
* **Можно установить пакетную цену (за весь проект или за папку)?**\
  Пока такой возможности нет, но она появится в следующих обновлениях.
* **Можно продавать печатные фото?**\
  Продажа фотографий осуществляется только в цифровом виде.\
  Для продажи печатной продукции [подключите типографию](photoprint.md) или [добавьте товары](products.md) в галерею.
*   **Можно установить разную цену на фото?**\
    Для одной галереи устанавливается единая цена за одну фотографию. Если необходимо, настройте систему скидок.

    В разных галереях можно задать разные цены на фотографии.
* **Можно добавить в корзину фото из разных галерей?**\
  Пока такой возможности нет. Для разных галерей будут разные корзины.
* **Можно настроить разный размер для скачивания?**\
  Фотографии можно скачать в оригинальном размере и в версии для соцсетей, если она включена в разделе **«Настройки диска»**.
* **У клиента сохранится доступ к скачиванию фото?**\
  Да, клиенту после успешной оплаты на почту автоматически отправляется ссылка для скачивания фотографий.
* **Как долго фото будут доступны для скачивания?**\
  Ссылка для скачивания фотографий действительна до тех пор, пока галерея не удалена с облачного диска.
* **Как сделать одновременно список избранного и корзину?**\
  Так сделать не получится. Можно создать 2 разных проекта или сначала воспользоваться функционалом для списков избранного, а затем сменить тип проекта для продажи фотографий.\
  Клиент сможет создать корзину из отобранных фотографий и оплатить их.
