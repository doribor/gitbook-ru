# Скачивание файлов в галерее

По умолчанию скачивание файлов включено. Если его требуется отключить, посмотрите раздел [#kak-otklyuchit-skachivanie](download.md#kak-otklyuchit-skachivanie "mention").

{% hint style="success" %}
Мы подготовили краткое [**видео-руководство**](https://t.me/wfolio/462) о скачивании фотографий в галерее — поделитесь ссылкой с клиентами.
{% endhint %}

### Оригинал и версия для социальных сетей

Фотографии можно скачать в двух вариантах:

* В оригинальном размере.
* В размере для социальных сетей. Она формируется для фото размером от 2561px по длинной стороне.

Чтобы включить формирование версии для соцсетей, перейдите в настройки диска.

### Сохранение фото на Яндекс Диск

Фотографии из галереи можно сохранить сразу на Яндекс Диск. Эта опция доступна, если все файлы в галерее — в формате JPEG.

{% hint style="info" %}
Если в галерее есть другие типы файлов (например, видео), ее не получится сохранить на Яндекс Диск. **Это техническое ограничение со стороны Яндекса.**
{% endhint %}

Включите сохранение на Яндекс в настройках диска.

### Поштучное скачивание и скачивание в архиве

Скачать файлы можно:

* **По одному** — нажав на иконку скачивания возле каждой фотографии или файла.
* **В архиве** — если нужно скачать сразу несколько файлов. Так можно скачать файлы из избранного, отдельную папку или всю галерею.

Архив загрузится в папку «Загрузки» или «Файлы» на устройстве. Перед просмотром файлов архив нужно распаковать.

### Как отключить скачивание

В настройках проекта (вкладка «Основные») найдите переключатель **«Разрешить скачивание оригиналов»**. Отключите его, чтобы запретить скачивание фото. Дополнительно можно настроить [водяной знак](../projects/protect.md#vodyanoi-znak).
