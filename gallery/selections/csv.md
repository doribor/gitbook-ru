# Данные об отборе в CSV

Для каждого проекта можно получить полезную информацию в документе формата CSV: сведения об отобранных фотографиях и иных файлах, ссылки на списки избранного или папки в проекте.

{% hint style="info" %}
Если после скачивания вместо данных видны странные символы, посмотрите инструкцию [csv.md](../../troubleshooting/csv.md "mention").
{% endhint %}

### Ссылки на списки избранного

С помощью этого документа можно быстро получить ссылки на заранее созданные списки для отбора. Удобно для передачи ссылок организаторам съемок или мероприятий (школьные, спортивные и др.).

Загрузите съемку и создайте нужные списки избранного. Для скачивания документа перейдите в раздел проекта **«Избранное»** и нажмите на **троеточие (⋮)**. Выберите **«Ссылки на списки в CSV»**. Скачается файл с именем _«проект\_links.csv»_.

<figure><img src="../../.gitbook/assets/csv selection link ru.png" alt=""><figcaption></figcaption></figure>

#### Список полей в документе

* `list_client` — имя клиента, создавшего список.
* `list_url` — ссылка на список.



### Ссылки на папки в проекте

С помощью этого документа можно быстро получить ссылки на все папки в проекте. Дополнительно можно скрыть все папки — так у каждого клиента будет доступ только к личной папке.

В проекте нажмите на **троеточие (⋮)** и выберите **«Список папок в CSV»**. Скачается файл с именем _«проект.csv»_.

<figure><img src="../../.gitbook/assets/csv folder link ru.png" alt=""><figcaption></figcaption></figure>

#### Список полей в документе

* `Folder` — название папки в проекте.
* `Url` — ссылка на папку.



### Список файлов в CSV

Содержит информацию о папке, в которой выбрали файл, список файлов и комментарии к списку и файлам. Удобно работать с отбором, если все папки в вашем проекте скрыты и вы направляете клиентам ссылку на отбор в личной папке.

В разделе проекта **«Избранное»** нажмите на **троеточие (⋮)** и выберите **«Список файлов в CSV»**. Скачается файл с именем _«проект\_files.csv»_.

<figure><img src="../../.gitbook/assets/pjorect file csv ru.png" alt=""><figcaption></figcaption></figure>

#### Список полей в документе

1. `file_folder` — название папки, в которой был выбран файл.
2. `file_name` — имя выбранного файла.
3. `file_added` — дата добавления файла в список.
4. `file_list` — название списка избранного (имя и фамилия клиента).
5. `list_comment` — комментарий к списку.
6. `file_comment` — комментарий к выбранным файлам.



### Сводный список файлов в CSV

Используется для передачи списков избранных файлов ретушерам или другим специалистам для дальнейшей обработки.

В разделе проекта **«Избранное»** нажмите на **троеточие (⋮)** и выберите **«Сводный список файлов в CSV»**. Скачается файл с именем _«проект\_summary.csv»_.

<figure><img src="../../.gitbook/assets/project summary csv ru.png" alt=""><figcaption></figcaption></figure>

#### Список полей в документе

1. `list_client` — имя клиента, создавшего список.
2. `list_name` — название списка (например, _«На ретушь»_, _«На печать»_).
3. `list_status` — статус списка (`In process` — в процессе отбора, `Completed` — отбор завершен).
4. `list_updated` — дата последнего обновления списка (добавление файлов, комментариев или изменение статуса).
5. `list_files` — имена выбранных файлов.
6. `list_comments` — комментарий клиента к списку и отдельным файлам.
7. `list_url` — ссылка на список избранного.



### Списки CSV для удаленных проектов

Если проект был удален, информация об отобранных файлах сохраняется в формате CSV. Перейдите в раздел диска **«Корзина»** и выберите **троеточие (⋮)** для скачивания списка файлов или сводного списка.

<figure><img src="../../.gitbook/assets/csv deleted project ru.png" alt=""><figcaption></figcaption></figure>



Эти списки позволяют удобно структурировать данные для работы с отбором фотографий, отправки ссылок клиентам и организации последующей обработки файлов.
