# Действия со списками

[Списки избранного](make-favorites-list.md) помогают удобно работать с отснятым материалом для выбора кадров для ретуши, печати или других задач. В этом разделе разберем, какие действия можно выполнять со списком избранного.

### Действия с одним списком

В разделе проекта **«Избранное»** нажмите на **троеточие (⋮)** возле нужного списка, чтобы вызвать меню действий для этого списка.

<figure><img src="../../.gitbook/assets/one list action menu ru.png" alt=""><figcaption></figcaption></figure>

* **Предпросмотр.** Позволяет посмотреть, как список избранного выглядит для клиента.
* **Настройки списка.** Здесь можно изменить название списка, добавить или убрать комментарии и настроить доступ.
* **Скопировать ссылку.** Получите ссылку, чтобы отправить клиенту готовый список избранного.
* **Скачать файлы.** Позволяет скачать все файлы из списка на компьютер в виде архива.
* [**Список файлов в CSV**](csv.md#spisok-failov-v-csv)**.** Экспортируйте список для передачи данных об отборе ретушерам или организаторам съемок.
* [**Lightroom & Capture One**](lightroom-and-capture-one.md)**.** Получите имена выбранных снимков для быстрого поиска по файлам и редактирования внутри этих программ.
* **Дублировать список.** Создает копию списка — полезно, если нужно внести изменения в список, но сохранить оригинальную версию.
* **Заказать печать.** Если подключена типография, можно отправить фотографии из списка [на печать](../../shop/gallery/photoprint.md#in-control-panel) прямо из галереи.

#### Управление доступом

* **Разрешить отбор.** Клиент сможет вносить изменения в список: добавлять или удалять фотографии.
* **Завершить отбор.** Закрывает возможность внесения изменений в список. Клиент сможет просматривать фотографии в списке, но не сможет их менять.

Скачивание файлов из списков доступно по умолчанию, если оно включено для всей галереи. Однако его можно запретить для галереи, но разрешить или запретить для отдельных списков, выбрав нужное действие:

* **Разрешить скачивание.** Включает возможность загрузки файлов клиентами.
* **Запретить скачивание.** Клиенты не смогут скачать файлы из списка.

#### <mark style="color:red;">Удаление</mark>

* <mark style="color:red;">**Удалить список и файлы.**</mark> <mark style="color:red;"></mark><mark style="color:red;">Полностью удаляет список и все файлы, которые в него входят. Восстановить их будет невозможно.</mark>
* <mark style="color:red;">**Удалить список**</mark> (но оставить файлы). Список избранного удаляется, но сами файлы остаются в проекте.



### Действия со всеми списками

В разделе проекта **«Избранное»** нажмите на **троеточие (⋮)** в правой части экрана, чтобы вызвать меню действий для всех списков проекта.

<figure><img src="../../.gitbook/assets/all lists action menu ru.png" alt=""><figcaption></figcaption></figure>

* **Скачать файлы.** Позволяет скачать все списки избранного с файлами в одном архиве.
* [**Lightroom & Caprute One**](lightroom-and-capture-one.md)**.** Получите имена выбранных снимков из всех списков для быстрого поиска по файлам и редактирования внутри этих программ.

#### Экспорт данных об отборе в CSV

Смотрите инструкцию: [csv.md](csv.md "mention").

#### Управление доступом для всех списков

* **Открыть все списки.** Разрешает вносить изменения в список для всех клиентов.
* **Завершить все списки.** Завершает отбор во всех списках. Клиенты не смогут добавлять фотографии в списки или удалять их.

#### <mark style="color:red;">Удаление всех списков</mark>

<mark style="color:red;">Действие</mark> <mark style="color:red;"></mark><mark style="color:red;">**«Удалить все списки»**</mark> <mark style="color:red;"></mark><mark style="color:red;">удалит списки и данные о выбранных фото.</mark> <mark style="color:red;"></mark><mark style="color:red;">**Отменить это действие невозможно.**</mark> Сами фотографии останутся в галерее.
