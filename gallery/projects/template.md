# Как сделать шаблон проекта

Шаблон проекта помогает быстро создавать новые галереи с готовой структурой папок, описаниями и ссылками — без необходимости настраивать всё с нуля.

### Как создать шаблон проекта

* **Создайте проект** на диске.
* Настройте его:
  * Настройте скачивание, параметры для отбора в списки избранное, пароль, контакты — всё, чем вы пользуетесь при передаче фото и что хотите скопировать.
  * Добавьте нужные папки, описания к папкам (добавление описания доступно после загрузки фотографий), задайте ссылки для каждой папки.
* В списке проектов нажмите на **три точки** **(⋮)** рядом с нужным проектом.
* Выберите **«Дублировать проект»**.

При этом:

* **Сохраняются** все настройки, структура папок и их ссылки.
* **Не переносятся** файлы внутри папок.

### Как закрепить шаблон

Вы можете закрепить до 3 проектов-шаблонов. Для этого в списке проектов нажмите нажмите на **три точки** **(⋮)** рядом с нужным проектом и выберите **«Закрепить проект»**.

Закрепленные проекты будут всегда отображаться в верхней части списка для быстрого доступа.
