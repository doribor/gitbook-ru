# Начало работы с диском

Раздел **«Облачный диск»** — встроенный в сайт сервис для передачи фотографий клиентам со страницей-визиткой с вашими контактами.

Ещё передаете фотографии через файлообменник? Тогда не удивляйтесь, почему у вас не работает сарафанный маркетинг.

Передача съемок через облачный диск на сайте позволит вам:

* Сделать передачу фотографий красивой.
* Развивать сарафанный маркетинг.
* Получать отзывы клиентов.

### Как это работает

Встроенный диск на вашем сайте — это не безликий файлообменник, а красивая галерея для вашего клиента. На диске есть все ваши контакты: телефон, мессенджеры, ссылки на социальные сети и сайт.

Просто загрузите фотографии и отправьте клиенту специальную ссылку для просмотра и скачивания.

**Клиенты с большой радостью будут делиться ссылкой со своими друзьями.**

На семейных, репортажных, свадебных или других групповых съемках это дает **взрывной рост посещаемости**.

Кроме этого, использование диска положительно сказывается на поисковом продвижении вашего сайта, так как он находится на адресе вашего сайта и посещаемость диска суммируется с посещаемостью сайта.

### Визитка

**Визитка** — это страница-мультиссылка с вашими контактами, по которым клиент может с вами связаться.

Визитка будет открываться:

* На тарифах «Сайт + Диск» — по адресу домена с добавлением `/me`, например: `mydomain.ru/me`.
* На тарифах «Только Диск» — по адресу вашего домена, например, `mydomain.ru`.

### Частые вопросы

* **Можно создавать защищенные галереи?**\
  Галереи на диске доступны только по прямой уникальной ссылке. Также дополнительно можно установить пароль для просмотра галереи.
* **Сможет ли мой клиент создать список избранных фотографий?**\
  Да, ваши клиенты смогут отбирать фотографии в галерее. С помощью «Избранного» легко сформировать список фото для книги, для ретуши, для слайдшоу и т.д.
* **Что можно загружать на диск?**\
  Поддерживается загрузка фотографий, видео и любых других файлов.
* **Можно сделать несколько папок в проекте?**\
  Да, можно.
* **А можно принимать заказ на печать фото или фотокниг?**\
  Да! Через облачный диск можно принимать заказы на печать. Подробнее смотрите тут: [photoprint.md](../../shop/gallery/photoprint.md "mention").
* **Есть ли мобильная версия для галереи?**\
  Да, галереи замечательно работают на всех устройствах.
* **Как собирать отзывы клиентов?**\
  После скачивания фотографий клиенту будет предложено оставить отзыв, который вы можете разместить у себя на сайте.
* **Можно добавить проект на другом языке?**\
  Да. Можно изменить язык проекта и вашей визитки в настройках проекта.
* **У меня не отображается превью файлов PNG и RAW в проекте. Почему?**\
  Превью изображений формата RAW любого размера, а также PNG с весом больше 10 МБ не отображаются на диске. Уменьшите изображение или загрузите изображения в формате JPEG. Для сжатия PNG можно использовать сайт [https://tinypng.com/](https://tinypng.com/).

Используйте облачный диск в своей работе, забудьте про безликие файлообменники!
