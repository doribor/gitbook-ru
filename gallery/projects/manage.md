# Управление проектами

В панели управления можно организовывать свои съемки, изменять их настройки, сортировать и закреплять важные съемки.

### Как поделиться проектом <a href="#share" id="share"></a>

Нажмите на иконку **«Поделиться»**. Скопируйте ссылку для отправки клиенту или скачайте QR-код. Или поделитесь галереей напрямую в мессенджере: WhatsApp, Telegram, Viber, ВК.

<figure><img src="../../.gitbook/assets/share project ru.png" alt=""><figcaption></figcaption></figure>

{% hint style="info" %}
Чтобы отправить ссылку в мессенджер, на вашем устройстве должно быть установлено **приложение** мессенджера.
{% endhint %}



### Действия с проектами <a href="#actions" id="actions"></a>

Нажмите на троеточие **(⋮)** возле проекта.

<figure><img src="../../.gitbook/assets/project actions ru.png" alt=""><figcaption></figcaption></figure>

Для каждого проекта доступны следующие действия:

* **Настройки.** Измените название, описание, настройки доступа и другие параметры.
* **Предпросмотр.** Откройте проект в режиме просмотра, чтобы увидеть, как он выглядит для клиента.
* **Закрепить.** Выделите до трех важных проектов, чтобы они всегда отображались вверху списка.
* **Дублировать.** Создайте копию проекта для удобного редактирования.
* <mark style="color:red;">**Удалить.**</mark> Переместите проект в корзину.\
  Восстановить его можно в течение 7 дней после перемещения в корзину, далее проект удаляется навсегда.

### Цветовые маркеры <a href="#mark" id="mark"></a>

Для удобной организации проектов можно использовать **цветовые маркеры** (до 5 цветов) — например, для разных типов съемок или приоритетных заказов.

Чтобы выделить проект цветом, нажмите на точку рядом.

### Сортировка проектов <a href="#sort" id="sort"></a>

Нажмите на кнопку **«Сортировать по»** над списком проектов.

<figure><img src="../../.gitbook/assets/sort by ru.png" alt=""><figcaption></figcaption></figure>

Проекты можно отсортировать:

* **По названию** (по алфавиту).
* **По размеру.**
* **По дате съемки.** Установить дату можно при создании проекта или в его настройках.
* **По дате создания.** По дате/времени создания проекта на облачном диске.
* **По дате истечения.** По дате, когда заканчивается срок хранения проекта.
* **По маркеру.** Группировка по цветовым меткам.
* **По скачиваниям.** По количеству скачиваний архивов из галереи.\
  Учитываются скачивания всей галереи, отдельных папок или списков избранного.\
  Не учитываются скачивания отдельных файлов в галерее — по ним есть статистика внутри проекта.

Выберите параметр сортировки и вариант отображения файлов:

* **По возрастанию.** От первой буквы алфавита к последней, от меньшего к большему, от ранних съемок к поздним, от менее популярных к более популярным.
* **По убыванию.** От последней буквы алфавита к первой, от большего к меньшему, от поздних съемок к ранним, от более популярных к менее популярным.
