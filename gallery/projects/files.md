# Какие файлы можно загружать на диск

### Передача фото и видео <a href="#photos-and-videos" id="photos-and-videos"></a>

Загружайте **фотографии** на диск в следующих форматах:

✔ **JPEG**.\
✔ **PNG** — до 20 МБ.\
✔ С цветовым профилем **sRGB** — это важно для корректного отображения фотографий после загрузки.

* **Можно загружать оригиналы в RAW (ARW, CR2, CR3 и др.), HEIC?**\
  Да, можно. Поддержка отображения таких файлов доступна в тарифах с безлимитным диском. В других тарифах они не будут показаны как фотографии — только как файлы.&#x20;

Для передачи **видео** загружайте файлы в форматах:

✔ **MP4**.\
✔ **MOV**.\
✔ С кодеком **h.264** **(AVC)** или **h.265 (HEVC)**.

{% hint style="info" %}
#### Воспроизведение видео

**Видеообложка** в галерее всегда воспроизводится без звука.

Любое **видео до 300 МБ** можно открыть и посмотреть в галерее. Ещё для таких видео можно настроить автовоспроизведение — тогда они будут автоматически проигрываться без звука при просмотре галереи. По клику на видео оно откроется в полноэкранном просмотре, где его можно посмотреть со звуком.

Остальные **видео (больше 300 МБ)** показаны со статичными обложками. Для просмотра их нужно скачать на устройство.
{% endhint %}



### Передача файлов <a href="#files" id="files"></a>

На диск можно загружать файлы любых форматов. Максимальный размер одного файла — 25 ГБ. После загрузки появится иконка файла и его формат.

Для просмотра файлов их необходимо скачать на устройство.

#### Ссылка на файл <a href="#file-link" id="file-link"></a>

Для всех фото, видео и файлов до 300 МБ формируется **ссылка на скачивание**. Если перейти по ней, файл автоматически скачается на устройство.

Такую ссылку можно добавить для [действия после оплаты](../../shop/general/add-product.md#actions) товара, чтобы передавать материалы (например, PDF-файл) клиентам.

<figure><img src="../../.gitbook/assets/copy file link ru.png" alt="" width="563"><figcaption></figcaption></figure>





{% hint style="info" %}
#### Пространство на диске для видео и файлов

На тарифах с безлимитными дисками, «Диск 240 ГБ», «Диск 1000 ГБ» выделено 120 ГБ для **видео** и **файлов других форматов (не JPEG/PNG)**.
{% endhint %}
