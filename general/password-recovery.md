# Обновление системы входа: теперь без пароля

Раньше при регистрации можно было придумать пароль, но для входа всегда требовалось дополнительное подтверждение — код доступа на почту. Это обеспечивало защиту аккаунта, но делало процесс входа сложнее.

Теперь пароль больше не нужен — вход выполняется только через код, который приходит на вашу почту. Это сделало процесс авторизации понятнее и удобнее.

#### Где получать код?

Код приходит на почту, указанную в разделе **«Учётная запись»** в настройках.
