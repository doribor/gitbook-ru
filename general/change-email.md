# Уведомления

Уведомления с вашего сайта по умолчанию отправляются на почту, указанную при регистрации. Они приходят, когда:

* Заполнена форма из блока «Контакты».
* Заполнена заявка на товар или оплачена покупка.
* Создан или изменен список избранного.
* Удален проект на диске.

Можно изменить адрес почты для уведомлений и не менять основной адрес почты (логин). Для этого перейдите в раздел настроек **«Уведомления»**.

Укажите новый адрес электронной почты для уведомлений. Если нужно, подключите дополнительно уведомления в Telegram.
