# Виджет «Вопрос/ответ»

Используйте виджет для красивого оформления списка частых вопросов и ответов для клиентов.

### Как добавить

* Откройте [составную страницу](../pages/custom-page.md) в панели управления и нажмите на кнопку <mark style="background-color:blue;">**(+)**</mark>, чтобы добавить новый блок.

<figure><img src="../../.gitbook/assets/add block again ru.png" alt=""><figcaption></figcaption></figure>

* Перейдите во вкладку **«Виджеты»** и выберите подходящий блок **«Вопрос/ответ»**.

<figure><img src="../../.gitbook/assets/add faq widget ru.png" alt="" width="563"><figcaption></figcaption></figure>

* Заполните список вопросов и ответов на них.
