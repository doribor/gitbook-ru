# Как создать QR-код для страницы

Вы можете создавать QR-коды для страниц сайта с помощью сервиса для генерации QR. Примеры сервисов:

* **Qrcode.tec** ([https://qrcode.tec-it.com](https://qrcode.tec-it.com/ru)).
* **Qrcode-monkey** ([https://www.qrcode-monkey.com/ru](https://www.qrcode-monkey.com/ru/)).
* **QR.io** ([https://qr.io](https://qr.io/)).

{% hint style="success" %}
Чтобы поделиться QR для галереи на диске, смотрите эту [инструкцию](../../gallery/share/qr.md).
{% endhint %}

### Как создать QR-код для страницы

* **Скопируйте ссылку** на страницу сайта.

<figure><img src="../../.gitbook/assets/copy page link 1 ru.png" alt=""><figcaption></figcaption></figure>

* Откройте любой из сервисов для генерации QR-кода. Вставьте ссылку в поле и скачайте код.
