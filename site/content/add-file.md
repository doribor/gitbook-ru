# Скачивание файлов через сайт

На сайт можно добавить файл для скачивания и передать клиенту гайд, прайс-лист, сертификат, руководство, архив и т.д.

{% hint style="success" %}
✔ Формат файла — любой.\
✔ Максимальный размер для получения ссылки на скачивание — 300 МБ.
{% endhint %}



* Создайте проект на облачном диске и загрузите в него файл для скачивания.
* Выберите иконку **«Скопировать ссылку»** в верхней панели над файлом.

<figure><img src="../../.gitbook/assets/copy file link ru.png" alt="" width="563"><figcaption></figcaption></figure>

* На [составной странице](../pages/custom-page.md) сайта добавьте блок с кнопкой, для этого нажмите на кнопку <mark style="background-color:blue;">**(+)**</mark>. Выберите блок из вкладки **«Кнопки»**. Перейдите в настройки кнопки.

<figure><img src="../../.gitbook/assets/button settings2.png" alt=""><figcaption></figcaption></figure>

Есть два способа настроить скачивание файла:

1. Сразу после клика на кнопку.
2. После заполнения контактной формы.

### Скачивание файла по кнопке

* В настройках кнопки выберите действие при клике **«Перейти на внешний адрес»**.
* Вставьте скопированную ссылку на файл для скачивания и сохраните изменения.

<figure><img src="../../.gitbook/assets/download file button ru.png" alt="" width="563"><figcaption></figcaption></figure>

Скачивание файла начнется после нажатия на кнопку.

<figure><img src="../../.gitbook/assets/download file example 1 ru.png" alt=""><figcaption></figcaption></figure>

### Скачивание после заполнения формы

Этот вариант позволяет собирать контактные данные клиентов перед скачиванием файла. После заполнения формы письмо с данными клиента будет направлено вам на почту.

* В настройках кнопки выберите действие **«Открыть товар/услугу»** и нажмите на **«Добавление и редактирование товаров»**.

<figure><img src="../../.gitbook/assets/dowbload file button add card ru.png" alt="" width="563"><figcaption></figcaption></figure>

* В открывшемся окне нажмите **«Добавить товар»**.
* Напишите название и описание услуги. В способе оплаты выберите **«Заявка (без указания цены)»**.

<figure><img src="../../.gitbook/assets/choose request ru.png" alt="" width="563"><figcaption></figcaption></figure>

* Во вкладке **«Настройки формы заказа»** настройте поля для заполнения формы (например, почта и телефон).

<figure><img src="../../.gitbook/assets/product card order form ru.png" alt="" width="563"><figcaption></figcaption></figure>

* Во вкладке **«Действия после заказа»** выберите переход на внешний адрес и вставьте скопированную ссылку на файл. Нажмите на кнопку **«Добавить»**.

<figure><img src="../../.gitbook/assets/add link for downloading ru.png" alt="" width="563"><figcaption></figcaption></figure>

* Вернитесь в настройки кнопки и выберите созданную карточку товара.

<figure><img src="../../.gitbook/assets/download file add file card ru.png" alt="" width="563"><figcaption></figcaption></figure>

После заполнения формы с контактными данными начнется скачивание файла.

<figure><img src="../../.gitbook/assets/download file example 2 ru.png" alt=""><figcaption></figcaption></figure>
