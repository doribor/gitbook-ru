# Начало работы с сайтом

Быстро запустить свой сайт — легко, даже если у вас нет опыта в создании сайтов.

Наша инструкция поможет вам создать сайт, который будет работать.

{% embed url="https://vimeo.com/566523688" %}

**Ваш сайт готов!**

Теперь подключите [персональный домен](../../domain/new.md) и поделитесь ссылкой на сайт в социальных сетях. Вы также можете передавать съемки клиентам с помощью [**облачного диска**](../../gallery/projects/easy-start.md) и развивать сарафанный маркетинг.

{% hint style="success" %}
Если у вас остались вопросы — напишите нам в чат поддержки на сайте. Мы с удовольствием вам поможем.
{% endhint %}
