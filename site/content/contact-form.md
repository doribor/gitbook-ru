# Блок с контактной формой

Блок **«Контакты»** позволяет посетителям сайта связаться с вами через форму обратной связи. Это удобный способ получать вопросы напрямую с сайта и установить первичный контакт с клиентом.

<figure><img src="../../.gitbook/assets/site forms example.png" alt=""><figcaption></figcaption></figure>

Для получения заказов, бронирования или оплаты лучше использовать **формы заказа**. Смотрите инструкции [request.md](../../shop/general/request.md "mention")и [add-product.md](../../shop/general/add-product.md "mention").



### Как добавить контактную форму на сайт

В режиме редактирования страницы нажмите на <mark style="background-color:blue;">**(+)**</mark>, чтобы создать новый блок. Выберите вкладку **«Контакты»**.

<figure><img src="../../.gitbook/assets/site forms ru.png" alt=""><figcaption></figcaption></figure>

В контактной форме есть три поля:

* Имя.
* Однострочное поле для сбора контактов — можно запросить почту, телефон или ссылку на соцсеть.
* Большое поле для сообщения посетителя сайта.

<i class="fa-location-question">:location-question:</i> В будущих обновлениях список полей будет расширяться.



#### Настройка текста в полях

Можно изменить текст подсказок для каждого поля в форме: например, сделать их более персонализированными. Для этого измените текст в нужном поле.

<figure><img src="../../.gitbook/assets/edit contact form fields ru.png" alt=""><figcaption></figcaption></figure>

Если на сайте несколько языков, подсказки для полей в форме переводятся **отдельно для каждого языка.** Это настраивается во вкладке с нужной языковой версией страницы. Подробнее про мультиязычные сайты читайте в статье [translate.md](../settings/translate.md "mention")



### Как отвечать на сообщения из формы

* Все сообщения из блока отправляются на вашу электронную почту.
* В панели управления Wfolio эти сообщения не сохраняются и не отображаются.

<i class="fa-lightbulb">:lightbulb:</i> Подключите [**уведомления в Telegram**](../../general/change-email.md), чтобы информация о новых сообщениях приходила туда. Так вы не пропустите ни одного обращения, даже если редко проверяете почту.



Чтобы связаться с клиентом, откройте письмо, полученное с сайта, и используйте контакты, которые указал клиент в форме. Ответьте удобным способом: по почте, в мессенджере или звонком.

{% hint style="success" %}
Рекомендуем проверять почту и Telegram-уведомления регулярно, чтобы не пропустить новые сообщения и быстро связаться с клиентом.
{% endhint %}



### Как добавить информацию об обработке персональных данных в форму

Чтобы посетитель сайта мог поставить галочку о согласии на обработку персональных данных, опубликуйте политику конфиденциальности в разделе настроек **«Правовые настройки»**. Подробнее смотрите тут: [privacy-policy.md](../../legal/privacy-policy.md "mention")
