# Как защитить видео от скачивания

Чтобы защитить видео от скачивания, загрузите их на специальный видеохостинг. Есть разные хостинги, но обычные, как YouTube, не предоставляют защиту от скачивания — есть много плагинов для скачивания или иного доступа к видео.

Ниже расскажем про сервисы, которые обеспечат высокую защиту от скачивания видео благодаря DRM-шифрованию.

#### Что такое DRM-шифрование

Это технология защиты, которая позволяет смотреть видео только при наличии специальных ключей. Благодаря ей нельзя копировать и использовать видео, даже если файл смогут скачать — без этих ключей видео не откроется или будет недоступно для воспроизведения.

{% hint style="info" %}
Сервисы ниже не вставляют рекламу в ваши видео.
{% endhint %}

### **Kinescope** ([kinescope.ru](https://www.kinescope.ru/))

Платформа предоставляет решения для хостинга видео, онлайн-трансляций, аналитики и защиты контента.

* [Тарифы](https://kinescope.ru/pricing)\
  Есть бесплатный тариф. Опция шифрования доступна на платном тарифе.
* Способы защиты видео:
  * DRM-шифрование. [Подробнее](https://kinescope.notion.site/DRM-35065319edb54b8083c25328a59b2797)
  * Водяные знаки на видео.
  * Ограничение доступа по доменам. Разрешите встраивание и воспроизведение видео только на вашем сайте.

{% hint style="success" %}
Видео с Kinescope можно добавить по прямой ссылке: [add-video.md](add-video.md "mention").
{% endhint %}



### **Boomstream** ([boomstream.com](https://boomstream.com/ru/))

С помощью сервиса можно продавать вебинары, тренинги — перед просмотром видео клиенту в всплывающем окне предложат оплатить услугу. Есть возможности для онлайн-трансляций.

* [Тарифы](https://boomstream.ru/pricing)\
  DRM-шифрование видео подключается и оплачивается отдельно, стоимость можно уточнить после регистрации на платформе.
* Подробнее [о защите видео на Boomstream](https://boomstream.ru/documentation/users/boomstream-platform-video-protection)
* Как получить HTML-код для вставки, [инструкция Boomstream](https://boomstream.ru/documentation/users/public-pages)
