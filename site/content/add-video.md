# Видео на сайте

Видео — отличный способ сделать сайт более привлекательным и информативным.

Можно <mark style="background-color:green;">загрузить видео напрямую с устройства</mark> или [встроить видео по ссылке](add-video.md#kak-vstroit-video-po-ssylke).

### Какие видео поддерживаются

✔ Размер — до 300 МБ.

✔ Формат — MP4 или MOV.

✔ Горизонтальные и вертикальные.

✔ Кодек — h.264 (AVC) или h.265 (HEVC).

{% hint style="info" %}
Это подходит для видео в блоках. В обложки страниц можно загружать видео до 100 МБ.
{% endhint %}

## Загрузка видео

Видео можно загрузить в любой блок на составной странице, а также на обложку страницы.

В блоке [**«Баннер»**](banner.md) и в [**обложке страницы**](../pages/page-cover.md) видео проигрываются автоматически и **всегда без звука**.

Для просмотра со звуком загрузите видео напрямую в подходящий блок: **«Галерея»,  «Фото и видео»,** **«Фото и видео с текстом», «Контакты», «Кнопки» и «Виджеты»**.

* Создайте или откройте составную страницу, пост в блоге или анонс.
* Нажмите на кнопку <mark style="background-color:blue;">**(+)**</mark>, чтобы добавить новый блок, и выберите блок из нужной вкладки.

<figure><img src="../../.gitbook/assets/add block template ru.png" alt=""><figcaption></figcaption></figure>

<figure><img src="../../.gitbook/assets/all vid blocks (1).png" alt=""><figcaption></figcaption></figure>

* Загрузите видео.

Для видео можно настроить автовоспроизведение — оно будет запускаться на странице автоматически без звука.

Как включить автовоспроизведение:

* в блоке **«Галерея»**: нажмите иконку **«Автозапуск (▷)»** в панели управления.

<div align="center"><figure><img src="../../.gitbook/assets/autoplay vid gallery block.png" alt=""><figcaption></figcaption></figure></div>

* в других блока&#x445;**:** наведите курсор на видео, откройте настройки и нажмите на переключатель **«Автозапуск»**.

<figure><img src="../../.gitbook/assets/autoplay vid separate ru.png" alt="" width="375"><figcaption></figcaption></figure>

### Как загрузить видео в альбом

В альбом можно встроить видео по ссылке. Чтобы загрузить видео напрямую, [преобразуйте альбом в составную страницу](add-blocks-to-album.md).

### Как встроить видео по ссылке

Более крупные видео можно встроить на сайт по ссылке. Можно добавить на сайт видео, размещенные на **YouTube**, **Vimeo**, **Kinescope**, **ВКонтакте** или **Rutube**.

Видео по ссылке можно встроить&#x20;

* в альбом:

<figure><img src="../../.gitbook/assets/add video album ru.png" alt=""><figcaption></figcaption></figure>

* на составную страницу — выберите блок из вкладки **Видео**:

<figure><img src="../../.gitbook/assets/embed vid ru.png" alt=""><figcaption></figcaption></figure>

**Как загрузить клип ВК / вертикальное видео из Rutube?**\
Эти сервисы не поддерживают добавление клипов на сторонние ресурсы, поэтому нет возможности вставить их по ссылке. Видео до 300 МБ вы можете загрузить напрямую на сайт.\
Клип ВК также можно добавить на составную страницу, встроив через HTML-код: [html-code.md](../widgets/html-code.md "mention")

## Идеи для портфолио с видео

* Создайте отдельную составную страницу с произвольной комбинацией блоков, включая видео.
* Организуйте галерею альбомов, где в каждом альбоме будет по одному видео.
* Используйте простой вариант — страницу альбома, содержащую только видео.
