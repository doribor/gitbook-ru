# Блок «Баннер»

Баннер — это широкоформатный блок, который можно использовать для привлечения внимания к важной информации или создания визуального акцента. Баннер может содержать изображение, несколько изображений или видео.

{% hint style="success" %}
#### Требования к фото и видео

**Для фото:**

✔ Рекомендуемый размер — не менее 2560×1920px.

**Для видео:**

✔ Размер — до 300 МБ.\
✔ Формат — MP4 или MOV.\
✔ Кодек — h.264/AVC или h.265/HEVC.
{% endhint %}



* На составной странице или лендинге нажмите на <mark style="background-color:blue;">**(+)**</mark>, чтобы добавить новый блок.

<figure><img src="../../.gitbook/assets/add block to a page ru.png" alt=""><figcaption></figcaption></figure>

* Во вкладке **«Баннеры»** выберите нужный баннер.

<figure><img src="../../.gitbook/assets/add banner ru.png" alt="" width="563"><figcaption></figcaption></figure>

* Загрузите фотографии или видео.
* Если нужно, настройте высоту баннера. Изображение или видео будут кропаться сверху и снизу.

<figure><img src="../../.gitbook/assets/banner height ru.png" alt=""><figcaption></figcaption></figure>

{% hint style="info" %}
Настройки высоты баннера применяются для компьютерной версии сайта. На телефонах баннер будет показан с фиксированной высотой. Это связано с различием в форматах экранов (очень широкий для ПК и небольшой по ширине для телефонов).
{% endhint %}

#### Эффект параллакса

Для **баннера-изображения** можно включить эффект параллакса — фото или видео будут двигаться вместе с прокруткой страницы. Для этого загрузите изображение в блок и активируйте переключатель **«Параллакс»**.

<figure><img src="../../.gitbook/assets/parallax for banners ru.png" alt=""><figcaption></figcaption></figure>

{% hint style="info" %}
Эффект параллакса доступен для всех [тем сайта](../design/theme.md), кроме Deka и Plain.

Он отображается при просмотре страницы **на компьютере**. Такой эффект для телефонов появится в следующих обновлениях.
{% endhint %}
