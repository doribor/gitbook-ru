# Ссылки на сайте

Вы можете добавить ссылку на **текст, изображение** или **кнопку**, чтобы посетители могли переходить на другие страницы вашего сайта или внешние сайты. Также можно сделать переход на внешний адрес в **меню** или создать **якорную ссылку** для быстрого перехода к нужному месту на странице.

{% hint style="info" %}
Чтобы добавить ссылку на кнопку — смотрите инструкцию [button-settings.md](button-settings.md "mention").
{% endhint %}



### Ссылка в тексте <a href="#text" id="text"></a>

* Откройте страницу и выделите текст, который нужно сделать ссылкой.
* В появившейся панели выберите иконку **«Вставить ссылку»**.

<figure><img src="../../.gitbook/assets/select text for link ru.png" alt=""><figcaption></figcaption></figure>

* Вставьте ссылку на страницу вашего сайта или другой адрес (например, [https://portfolioaward.com](https://portfolioaward.com/)). Сохраните изменения.

<figure><img src="../../.gitbook/assets/add link text ru.png" alt=""><figcaption></figcaption></figure>

[Как добавить активный номер телефона?](messenger-links.md)

Cсылки на ваш сайт открываются в той же вкладке, а переходы на другие сайты — в новой вкладке браузера.

{% hint style="success" %}
#### Где можно сделать ссылку в тексте

✔ На страницах сайта.\
✔ В описании проекта на [облачном диске](../../gallery/projects/easy-start.md).\
✔ В описании [товара](../../shop/general/add-product.md) в «Магазине».\
✔ В письме о подтверждении покупки товара.
{% endhint %}

### Ссылка на изображении <a href="#image" id="image"></a>

Ссылку можно вставить на [составной странице](../pages/custom-page.md) для фотографий из блоков «Изображение».

* Откройте страницу и нажмите на изображение. В панели инструментов выберите **«Ссылка»**.

<figure><img src="../../.gitbook/assets/add link to image ru.png" alt=""><figcaption></figcaption></figure>

* Настройте действие:
  * **«Открыть страницу сайта»** — если нужно сделать переход на другую страницу сайта.
  * **«Перейти на внешний адрес»** — если ссылка ведет на другой сайт.
  * Или выберите другое действие — можно добавить номер телефона для звонка, настроить переход в социальные сети/мессенджеры или открыть [товар](../../shop/general/add-product.md) из магазина.

<figure><img src="../../.gitbook/assets/add link image ru.png" alt="" width="563"><figcaption></figcaption></figure>

* Вставьте ссылку и сохраните изменения.

<figure><img src="../../.gitbook/assets/external site image link ru.png" alt="" width="563"><figcaption><p>Так можно добавить ссылку на другой сайт</p></figcaption></figure>



### Ссылка в меню <a href="#menu" id="menu"></a>

В меню показаны ссылки на страницы вашего сайта. Но можно добавить ссылку на другой сайт или на ваш номер телефона.

* Нажмите на **«Добавить на сайт»** и выберите ссылку.

<figure><img src="../../.gitbook/assets/add link ru.png" alt="" width="323"><figcaption></figcaption></figure>

* Дайте ссылке название, которое будет в меню, и включите отображение в меню сайта.
* Добавим телефон в меню: выберите действие **«Сделать звонок»** и введите номер телефона.

<figure><img src="../../.gitbook/assets/add phone menu link ru.png" alt="" width="563"><figcaption></figcaption></figure>



### Якорная ссылка (переход к блоку на странице) <a href="#anchor" id="anchor"></a>

Каждому блоку на сайте присвоен **уникальный идентификатор** (якорь), который можно использовать для перехода к нужному месту на странице.

* Откройте [составную страницу](../pages/custom-page.md) и перейдите в настройки блока.
* В открывшемся окне найдите **якорную ссылку** (она начинается с `#section`). Нажмите на нее, чтобы скопировать.

<figure><img src="../../.gitbook/assets/get anchor link ru.png" alt=""><figcaption></figcaption></figure>

* Выделите текст, изображение или кнопку, на которые хотите добавить ссылку. Или добавьте якорную ссылку в меню по инструкциям выше.
* Сохраните изменения на странице.

{% hint style="success" %}
При нажатии на якорную ссылку посетитель попадет сразу на нужный блок страницы.
{% endhint %}
