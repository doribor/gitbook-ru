# Блок «Слайдер»

Чтобы создать динамичную галерею с перелистыванием изображений, используйте блоки-слайдеры. Они отлично подойдут для демонстрации нескольких лучших кадров на главной странице — для презентации портфолио, товаров или атмосферы вашего проекта.



* В разделе **«Сайт»** нажмите на кнопку **«Добавить на сайт»**&#x200E; и выберите составную страницу.
* Введите название страницы и нажмите на **«Добавить»**.

<figure><img src="../../.gitbook/assets/add custom page ru.png" alt="" width="323"><figcaption></figcaption></figure>

* При наполнении страницы выберите пункт **«Составить самостоятельно»**. Нажмите на кнопку <mark style="background-color:blue;">**(+)**</mark> для добавления нового блока.
* В появившемся окне откройте вкладку **«Слайдеры»** и добавьте подходящий блок:
  * **Слайдер** — в блоке показана одна фотография, они сменяются одна за другой.
  * **Слайдер-лента** — несколько фотографии идут друг за другом.
  * **Слайдер-сетка** — фотографии квадратные, переключается несколько фото в блоке за раз.

<figure><img src="../../.gitbook/assets/choose slider ru.png" alt=""><figcaption></figcaption></figure>

* Загрузите фотографии в слайдер.
* Настройте скорость анимации в слайдере и расположите фотографии в нужном вам порядке, перетащив их мышкой.

{% hint style="info" %}
Для слайдера и слайдера-ленты можно настроить **случайный порядок фотографий**.
{% endhint %}
