# Обложка для ссылки на сайт

Обложка отображается рядом со ссылкой на ваш сайт. Также это изображение называют «картинка к внешним ссылкам», «превью» или «сниппет».

Размер обложки зависит от социальной сети. Чтобы сделать обложку, подходящую для Facebook и ВКонтакте, используйте горизонтальное изображение размером **не менее 1200px**. При этом ваше изображение будет уменьшено до указанного размера и кадрировано по высоте **до 630px** для Facebook и **до 480px** для ВК.

### Обложка сайта для соцсетей

Чтобы загрузить обложку сайта (главной страницы) для соцсетей перейдите в раздел **«Настройки» → «Настройки сайта»** и, в самом низу страницы, загрузите изображение в блоке **«Обложка для социальных сетей»**.

<figure><img src="../../.gitbook/assets/site cover snippet ru.png" alt="" width="563"><figcaption></figcaption></figure>

{% hint style="warning" %}
Обложка сайта в социальных сетях обновится через некоторое время, когда обновится кэш в социальной сети. Чтобы не ждать, [обновите кэш страницы](../pages/clear-cache.md).
{% endhint %}
