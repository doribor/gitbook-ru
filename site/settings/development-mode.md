# Режим разработки сайта

По умолчанию ваш сайт доступен для посетителей. Чтобы закрыть его, перейдите в **«Настройки» → «Настройки сайта»** и нажмите на кнопку «Закрыть сайт».

<figure><img src="../../.gitbook/assets/site development mode ru.png" alt="" width="563"><figcaption></figcaption></figure>

Теперь на главной странице сайта будет отображаться ваша визитка с контактами. Информацию из визитки можно отредактировать в разделе [**«Визитка»**](../../contacts.md).
