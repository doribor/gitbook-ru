# Поддержка темной темы в браузере

На разных устройствах и браузерах есть возможность установки **темного режима (dark mode)** просмотра:

* На устройствах Android или Widows включается в настройках браузера.
* На устройствах iOS включается в настройках устройства.

### **Как работает поддержка темной темы**

**Если у вас светлый фон сайта**, а у посетителя сайта включена темная тема, то цвет фона на экране посетителя будет **изменен на черный**, а цвет текста — на белый.

**Если у вас темный фон сайта**, то он будет отображаться для посетителей без изменений.

Для включения адаптации под темную тему браузера перейдите на страницу **Дизайн** в раздел **Цвета сайта**, выберите пункт **«Поддержка темной темы»**.

{% hint style="info" %}
**Эта настройка применяется для показа сайта на устройствах с темным режимом в браузере.** Она не изменяет цвета на вашем сайте полностью. Для изменения цветовой схемы сайта выберите в разделе «Цвета сайта» нужную вам цветовую палитру.
{% endhint %}
