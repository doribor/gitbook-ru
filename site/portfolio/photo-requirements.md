# Требования к фотографиям

Загружайте фотографии в большом размере, сайт самостоятельно уменьшит их для отображения на разных устройствах без потери качества.

Рекомендуем загружать фотографии в формате **JPEG** размером не менее **2560px** по длинной стороне. Также вы можете загружать изображения в формате **PNG размером не более 10 МБ**.

Для сжатия изображения в формате PNG можно использовать сайт [**https://tinypng.com**](https://tinypng.com/).

{% hint style="success" %}
#### Цветовой профиль фотографий

Убедитесь, что у фотографий установлен цветовой профиль **sRGB**.

Это важно для корректного отображения фотографий после загрузки на сайт.
{% endhint %}
