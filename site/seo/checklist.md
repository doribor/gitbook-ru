# Чек-лист по оптимизации сайта

Чтобы сайт был на хороших позициях в поиске, вам не нужно платить деньги за продвижение или быть техническим специалистом. Достаточно оптимизировать сайт и поддерживать на нем активность — поиск будет продвигать ваш сайт естественным образом.

Команда Wfolio подготовила чек-лист для проверки оптимизации сайта. В конце каждого раздела есть ссылка для проверки каждого пункта. Если вы ответили «Нет» хотя бы один раз или сомневаетесь в ответе — перейдите по ссылке, и вы найдёте рекомендации по улучшению сайта с примерами и пояснениями от наших SEO-специалистов.



{% stepper %}
{% step %}
**Главная страница сайта**

* Написан корректный заголовок (H1)
* На странице есть уникальный текст (1000-1500 символов)
* Текст содержит абзацы и подзаголовки

[Как оптимизировать главную страницу сайта](../../checklist/seo/main-page.md)
{% endstep %}

{% step %}
**Настройки сайта**

* Название (title) сайта заполнено корректно
* Описание (description) сайта заполнено корректно
* В тексте внизу сайта (footer) указан регион работы

[Как заполнить главные настройки сайта для продвижения](../../checklist/seo/main-settings.md)
{% endstep %}

{% step %}
**Структура сайта**

* Есть страницы «Услуги», «Обо мне», «Отзывы»
* На страницах «Услуги», «Контакты» указан регион работы
* На страницах используются заголовки с ключевыми словами
* Тексты содержат абзацы и подзаголовки
* Страницы оформлены без грамматических ошибок

[Проверьте наполнение вашего сайта](../../checklist/seo/navigation.md)
{% endstep %}

{% step %}
**Обновление контента**

* Используются разделы «Анонсы» или «Блог»
* В названиях анонсов или постов в блоге есть ключевые слова
* В заголовках и текстах есть упоминание о месте съемки
* На страницах, кроме фотографий, есть тексты
* Новые страницы добавляются регулярно

[Как вести сайт, чтобы он продвигал вас в поиске](../../checklist/seo/updates.md)
{% endstep %}
