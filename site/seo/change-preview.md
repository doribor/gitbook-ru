# Как поменять обложку для ссылки на отдельной странице

Обложка для соцсетей (превью ссылки, сниппет) показывается рядом со ссылкой, когда вы делитесь ею:

<figure><img src="../../.gitbook/assets/link preview ru.png" alt="" width="563"><figcaption></figcaption></figure>

{% hint style="info" %}
Если нужно установить единую обложку для всех страниц сайта, замените ее в разделе **«Настройки» → «Настройки сайта»** в пункте **«Обложка для социальных сетей»**.
{% endhint %}



Изменить обложку для отдельной странице можно так:

* Откройте страницу в панели управления и перейдите в **«Настройки»**.
* Во вкладке **«SEO и соцсети»** загрузите новое изображение и сохраните изменения.

<figure><img src="../../.gitbook/assets/change link preview ru.png" alt="" width="563"><figcaption></figcaption></figure>

* После обновления [очистите кэш ссылки](../pages/clear-cache.md), чтобы изменения отобразились корректно.
