# Заголовки на страницах (H1, H2, H3)

Заголовки помогают структурировать страницу и делают ее понятной для посетителей и поисковых систем. На Wfolio есть три уровня заголовков: H1, H2 и H3. Вот как они работают.

### Основной заголовок (H1)

На каждой странице может быть только один заголовок H1. Он задается автоматически:

* Если на странице есть [обложка](../pages/page-cover.md), то основным заголовком будет надпись на фотографии.

* Если обложки нет, то H1 назначается первому заголовку на странице:
  * Заголовок, добавленный через текстовый редактор (стиль форматирования «Заголовок»).
  * Название страницы — например, если это альбом или пост в блоге.

{% endhint %}

### Заголовок (H2)

Все следующие заголовки, которые вы добавляете через стиль форматирования «Заголовок», будут заголовками второго уровня H2.

Используйте их для логичной структуры: разделяйте основные блоки информации, выделяйте смысловые части.

#### Подзаголовок (H3)

«Подзаголовки» в форматировании текста — это заголовки третьего уровня H3. Их можно выбрать вручную при редактировании текста: достаточно выделить текст и задать стиль заголовка третьего уровня.

### Советы по оформлению заголовков

* Используйте заголовки и подзаголовки, чтобы упростить навигацию по тексту.
* Включайте в заголовки ключевые слова, связанные с темой вашего сайта и страницы. Например, для продвижения в поиске в городе Калининград намного полезнее написать на главной странице заголовок _«Репортажный фотограф в Калининграде»_, чем _«Добро пожаловать!»_.
* Не делайте заголовками обычный текст — они нужны для структурирования текста на страницах, а не для визуального оформления. Настроить размер шрифта для заголовков, подзаголовков и обычного текста можно в разделе «Дизайн».
