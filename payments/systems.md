# Платежные системы

Чтобы посетители могли производить оплату онлайн, а вы могли принимать их платежи, необходимо подключить одну из доступных платежных систем и добавить товары или услуги для продажи.



{% include "../.gitbook/includes/payment-systems-list.md" %}

{% hint style="info" %}
Wfolio не занимается приемом и обработкой платежей, а только предоставляет возможность подключения внешних сервисов для выполнения данных задач. Перед подключением платежного сервиса изучите его тарифы и правила предоставления услуг.
{% endhint %}
