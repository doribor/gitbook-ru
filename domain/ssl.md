# SSL-сертификат и HTTPS

Благодаря **SSL-сертификату** сайт работает по безопасному HTTPS-протоколу. Этот протокол защищает данные посетителей сайта от перехвата.

Сайты на Wfolio получают SSL-сертификат автоматически. Это бесплатно.

Почему мы выдаем сертификат бесплатно:

* Данные посетителей сайта защищаются шифрованием. Это повышает безопасность.
* Браузеры отмечают сайт как надежный.
* Сайты с HTTPS-протоколом лучше ранжируются в результатах поиска.

Для доменов с адресом .wfolio.pro SSL-сертификат активен сразу.

{% hint style="info" %}
SSL-сертификат выдается автоматически после подключения домена к сайту. Это может занять **до 24 часов**.
{% endhint %}



### Частые вопросы

* **Как получить SSL-сертификат?**\
  Сертификаты выдаются автоматически и **бесплатно** для всех сайтов на Wfolio.
* **Я подключаю домен. Когда будет выдан SSL-сертификат?**\
  Для доменов `.wfolio.pro` сертификат активен сразу. Для остальных доменов сертификат активируется через несколько часов после подключения домена: в течение 3-х часов — если купили [домен на Wfolio](new.md), в течение 24-х часов — если подключили [сторонний домен](existing-domain.md).
* **Мой регистратор домена просит подключить услугу SSL-сертификата. Нужно соглашаться?**\
  Не нужно. Мы выдаем сертификат SSL бесплатно и автоматически. У регистратора домена это может быть временно бесплатная услуга. Позже она может стать платной. Поэтому советуем вам **отключить все дополнительные услуги у регистратора**.
