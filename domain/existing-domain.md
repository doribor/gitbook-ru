# Подключение существующего домена

Чтобы подключить домен, зарегистрированный у стороннего регистратора, выполните следующие шаги:

* Перейдите в раздел настроек **«Адрес сайта»** или **«Веб-адрес диска»**.
* Нажмите на ссылку **«У меня уже есть домен»**.
* В открывшемся окне введите адрес вашего домена и нажмите кнопку **«Подключить»**.
* Далее произведите настройку домена у вашего регистратора.

### Изменение подключенного домена <a href="#change-domain" id="change-domain"></a>

Если у вас уже подключен домен и вы хотите его сменить:

* Если вы регистрировали домен через Wfolio, то нажмите на кнопку <mark style="color:blue;">**«Изменить домен»**</mark> и подключите любой бесплатный домен `.wfolio.pro`.
* Если вы подключали домен у стороннего регистратора, нажмите кнопку <mark style="color:red;">**«Отключить домен»**</mark>.&#x20;

После этого перейдите по ссылке **«У меня уже есть домен»**. Далее произведите настройку домена у вашего регистратора.

## Настройка домена в личном кабинете регистратора <a href="#registrar-settings" id="registrar-settings"></a>

Настроить домен можно двумя способами.

### Подключение через DNS-серверы Wfolio <a href="#dns-records" id="dns-records"></a>

Этот способ подойдет, если вам не требуется управлять ресурсными записями домена (A, AAAA, MX, TXT) для создания поддоменов, подключения почты или других сервисов. После добавления DNS домен будет делегирован Wfolio — добавление новых записей, если потребуется, будет осуществляться через нашу поддержку.

Если вы ведете деятельность в России, рекомендуется настроить домен для работы с российскими серверами. Если вы работаете за пределами РФ, настройте подключение через европейские серверы.

* Укажите в настройках домена только два DNS-сервера Wfolio, значения которых указаны в таблице ниже.
* **Убедитесь, что все другие DNS-серверы удалены.** Это необходимо для корректного подключения домена.

### Подключение через A-записи для домена <a href="#a-records" id="a-records"></a>

Этот способ подойдет, если:

* Регистратор не позволяет изменять DNS-серверы.
* Вы хотите самостоятельно вносить изменения в ресурсные записи домена A, AAAA, MX, TXT (например, для подключения поддомена сайта, почты, другого сервиса).

Настроить работу домена можно на серверы в России, Европе (подойдет для работы сайта во всем мире), Казахстане или Беларуси.

Добавьте две A-записи с IP-адресом, ведущим на Wfolio:

1. Для первой записи укажите символ `@` в имени записи (хоста). Некоторые регистраторы требуют вместо символа `@` указать ваш домен, например, `example.com`. Если в поле с именем записи не удается указать ни символ `@`, ни имя домена, оставьте его пустым.
2. Для второй записи укажите имя записи (хост) `www`.

* **Убедитесь, что для домена указаны только две A-записи с именами `@` и `www`.** Если есть дополнительные записи для этих хостов, это может вызвать конфликт IP-адресов и привести к ошибке подключения.
* Если у домена есть AAAA-записи для `@` или `www`, их необходимо удалить.
