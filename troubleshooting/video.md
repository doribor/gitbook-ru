# Ошибки при добавлении видео

Иногда видео может не добавиться на сайт или не воспроизводиться — ниже рассказываем, что делать.



#### Видео ВК добавилось, но посетитель сайта не может его посмотреть. А я могу <a href="#vk-privacy" id="vk-privacy"></a>

Если посетитель сайта видит ошибку, как на скриншоте ниже, сложность может быть с настройками приватности для вашего профиля ВК.

<figure><img src="../.gitbook/assets/error vk video ru.png" alt="" width="563"><figcaption></figcaption></figure>

✔ Перейдите в настройки профиля: вкладка **«Приватность» → «Прочее»**. В настройке «Кому в интернете&#x20;видна моя страница» должно быть указано «Всем» — иначе посетитель сайта не сможет посмотреть видео без авторизации в ВК.

#### Видео добавилось, но оно меньше нужного размера. При воспроизведении видео размер нормальный <a href="#cover" id="cover"></a>

Обложка исходного видео слишком маленькая и не может быть отображена во всем блоке без потери качества.

✔ В настройках видео загрузите обложку большего размера.

#### Не добавляется клип ВК <a href="#vk-clip" id="vk-clip"></a>

ВКонтакте не позволяет добавлять на сайт клипы через ссылку, это ограничение их платформы.

✔ Клип можно встроить через HTML-код по инструкции [html-code.md](../site/widgets/html-code.md "mention"). Чтобы скопировать код клипа для вставки, поделитесь клипом и выберите вкладку **«Экспортировать»**. Код выглядит так:\
`<iframe src="https://vk.com/video...`\
`КОД ВИДЕО>`\
`</iframe>`.

#### Не добавляется шортс Rutube <a href="#rutube-shorts" id="rutube-shorts"></a>

Шортсы Rutube не получится добавить на другие сайты по ссылке — это ограничение Rutube.

✔ Клип можно встроить через HTML-код по инструкции [html-code.md](../site/widgets/html-code.md "mention"). Чтобы скопировать код клипа для вставки:

* Замените в ссылке "shorts" на "video". Вместо ссылки вида `https://rutube.ru/shorts...` будет ссылка `https://rutube.ru/video...`.
* Обновите страницу — откроется страница с видео в горизонтальном плеере.
* Нажмите **«Поделиться»** и выберите **«Код вставки плеера»**. Код выглядит так:\
  `<iframe width="..." height="..." src="https://rutube.ru/play...`\
  `КОД ВИДЕО>`\
  `</iframe>`.



{% hint style="success" %}
Скоро на сайт можно будет загружать видео напрямую с устройства, без ссылок.

Следите за нашими обновлениями в [**Telegram**](https://t.me/wfolio).
{% endhint %}
