# Создание второго сайта

Вы можете создать несколько сайтов под одной учетной записью.

Для этого наведите курсор на иконку профиля и выберите **«Создать еще сайт»**.

![](<../.gitbook/assets/add another site ru.png>)

{% hint style="info" %}
На второй и последующие сайты действует акция: первый месяц сайта за 1 рубль **или** скидка 1500 рублей на первую годовую оплату.

После оплаты для первого сайта будет начислено **3 подарочных месяца** (для всех тарифов, кроме [Яндекс 360 Премиум Wfolio](yandex.md)).
{% endhint %}
