# Рекламный пиксель

Пиксель — это инструмент аналитики, который позволяет отслеживать всех посетителей сайта. Как только человек заходит на страницу вашего сайта, пиксель автоматически запоминает этот визит и ID пользователя в социальной сети. Собрав аудиторию из посетителей сайта, вы сможете настроить гибкую рекламную кампанию по выбранной аудитории в социальных сетях.

Можно подключить два пикселя — пиксель ВК Рекламы или Facebook\* (Meta) Pixel.

{% hint style="info" %}
Пиксель можно установить на сайт, даже если вы не планируете запускать рекламу в ближайшее время. **Возможность в будущем сформировать необходимую аудиторию для проведения рекламной кампании** окажется очень полезной.
{% endhint %}

## Для чего и как использовать пиксель

Основная идея — собирать аудиторию, которая была на вашем сайте или диске, а потом эффективно и адресно делать для них рекламные предложения в социальных сетях. Несколько примеров использования информации, собранной с помощью пикселя:

* **Напоминать о себе**. Вы на регулярной основе сможете напоминать о себе пользователям, которые посещали ваш сайт, знают вас и лояльно относятся к вашему творчеству. Реклама для такой аудитории будет намного эффективнее обычной рекламы.
* **Собирать аудиторию из друзей клиентов.** Если вы используете [**облачный диск**](../gallery/projects/easy-start.md) для регулярной передачи снимков, а ваши клиенты делятся ссылкой с друзьями, то вы также сможете собирать их в отдельную аудиторию и подготавливать для них специальные предложения.
* **Посещение определенных страниц.** Вы сможете подготовить индивидуальные, акционные предложения для посетителей определенных страниц. Например, сделать рекламу с акцией для всех посетителей, которые посещали страницу услуг за последние 30 дней.
* **Сделать акцию ко дню рождения.** Сделать специальное предложение для пользователей, которые заходили на ваш сайт и у которых скоро будет день рождения.
* **Запустить догоняющую кампанию.** Делать догоняющие рекламные компании по конкретным товарам и услугам.  Например, посетитель интересовался пресетом или видеоуроком на вашем сайте, но не оставил заявку. Вы сможете показать ему в социальной сети рекламу именно с тем предложением, которым он интересовался.

## Как добавить пиксель ВКонтакте <a href="#vk" id="vk"></a>

* Зарегистрируйтесь в [**ВК Рекламе**](https://ads.vk.com/).
* Создайте пиксель и получите код пикселя для вставки на сайт по [инструкции ВК Рекламы](https://ads.vk.com/help/general/sites/pixel).
* Перейдите в панель управления Wfolio в раздел настроек **«Интеграция с сервисами»**.
* В поле **«Сторонний код в теге \<head>»** вставьте полученный код. Сохраните изменения.

<figure><img src="../.gitbook/assets/vk integration ru.png" alt="" width="563"><figcaption></figcaption></figure>

{% hint style="info" %}
Для подключения пикселя ВК **не нужно вставлять ID пикселя в раздел «Счетчики и пиксели»** — он подключается только через раздел «Интеграция с сервисами».
{% endhint %}

## Как добавить пиксель Meta (Facebook) <a href="#facebook" id="facebook"></a>

* Зарегистрируйтесь в [**Facebook Business**](https://business.facebook.com/overview) и создайте бизнес-портфолио для аккаунта. Укажите название компании, ваше имя и рабочий email.
* После регистрации в панели управления слева выберите [**Events manager**](https://business.facebook.com/events_manager2).
* Нажмите в окне слева на <mark style="color:green;">«</mark><mark style="color:green;">**Подключить данные»**</mark>.
* Выберите источник данных **«Интернет»** и нажмите на кнопку **«Далее»**.
* Дайте название набору данных, например, _«пиксель `адрес сайта`»_ и **создайте** пиксель. Окно с дальнейшими настройками можно закрыть.
* В списке источников данных **скопируйте ID пикселя**.

<figure><img src="../.gitbook/assets/meta facebook pixel ru.png" alt="" width="563"><figcaption></figcaption></figure>

* Перейдите в панель управления Wfolio в раздел настроек **«Счетчики и пиксели»**. **Вставьте скопированный ID** в поле «**Пиксель Facebook**» и сохраните изменения. При этом код для пикселя Facebook в разделе «Интеграция с сервисами» добавлять не нужно.

<figure><img src="../.gitbook/assets/pixel facebook connect ru.png" alt="" width="563"><figcaption></figcaption></figure>



{% hint style="warning" %}
**Пиксель устанавливается на весь сайт. Его нельзя установить на отдельные страницы.** Если нужно отслеживать действия на конкретных страницах, задайте цели в настройках пикселя.
{% endhint %}



\*деятельность организации Meta, владеющей Facebook, признана экстремистской и запрещена на территории РФ.
