# Сервисы аналитики

Сервисы аналитики — бесплатные инструменты аналитики/статистики, которые собирают данные о посещаемости вашего сайта и представляют их в виде удобных отчетов. Они отслеживают источники трафика и помогают оценивать эффективность вашего сайта и рекламы.

## Как подключить Яндекс Метрику <a href="#ya-metrika" id="ya-metrika"></a>

{% hint style="info" %}
Инструкция ниже — для пользователей из России, Беларуси и Казахстана.

Если вы находитесь в другом регионе, подключите Метрику, вставив **HTML-код** в раздел настроек **«Интеграция с сервисами»**.
{% endhint %}

* Авторизуйтесь на сайте [**Яндекс Метрики**](https://metrika.yandex.ru/) и добавьте [новый счетчик](https://metrika.yandex.ru/add). Введите имя счетчика и адрес вашего сайта. Ознакомьтесь с условиями сервиса и нажмите на **«Создать счетчик»**.

<figure><img src="../.gitbook/assets/ya metrika add counter ru.png" alt=""><figcaption></figcaption></figure>

*   Заполните профиль:

    * в поле **«CMS»** выберите «Другое» и ниже введите «Wfolio».
    * в поле **«CRM»** также выберите «Другое» → «Другое».

    Нажмите **«Продолжить»**.

<figure><img src="../.gitbook/assets/ya metrika profile.png" alt=""><figcaption></figcaption></figure>

{% hint style="success" %}
Заполнить профиль можно в настройках после создания счётчика. Чтобы пропустить заполнение профиля, просто нажмите **«Продолжить»,** не заполняя поля.
{% endhint %}

* Выберите вкладку **«CMS и конструкторы сайтов»** и скопируйте номер вашего счетчика. Затем внизу страницы нажмите на кнопку <mark style="background-color:yellow;">**«Начать пользоваться»**</mark>.

<figure><img src="../.gitbook/assets/ya metrika counter get (1).png" alt=""><figcaption></figcaption></figure>

* Перейдите в настройки Wfolio в раздел **«Счетчики и пиксели»**. Вставьте скопированный номер счетчика в поле **«Яндекс Метрика»** и сохраните изменения.

<figure><img src="../.gitbook/assets/ya metrika add wfolio.ru.png" alt="" width="563"><figcaption></figcaption></figure>

* Готово, счетчик установлен. Теперь будет собираться статистика о посещаемости вашего сайта. Данные можно увидеть в отчетах Яндекс Метрики.

{% hint style="info" %}
Номер ранее созданного счетчика в Метрике отображается в списке счетчиков или в любом отчете статистики.

<img src="../.gitbook/assets/find metrika counter ru.png" alt="" data-size="original">
{% endhint %}

#### Если подключаете Вебвизор в Метрике

**По умолчанию вебвизор отключен в мобильной версии сайта**. Вебвизор расходует большое количество оперативной памяти устройства. Это может негативно отразиться на скорости работы сайта для его посетителей, а на старых моделях телефонов Вебвизор может работать некорректно.

Чтобы добавить Вебвизор в мобильную версию, нужно удалить номер счетчика Яндекс Метрики из раздела настроек **«Счетчики и пиксели»**. После этого скопируйте полный код счетчика Метрики с включенным вебвизором. Вставьте код в раздел настроек **«Интеграция с сервисами»** в поле «Сторонний код в теге \<head>».

<figure><img src="../.gitbook/assets/metrika webvisor ru.png" alt="" width="563"><figcaption></figcaption></figure>

{% hint style="info" %}
Мы не рекомендуем оставлять Вебвизор включенным в мобильной версии на постоянной основе.
{% endhint %}



## Как подключить Google Analytics <a href="#google" id="google"></a>

* Выполните вход в Google и создайте новый аккаунт в сервисе [**Google Analytics**](https://analytics.google.com/). Заполните форму регистрации аккаунта. Введите **«Название аккаунта»**.

<figure><img src="../.gitbook/assets/create ga account ru.png" alt=""><figcaption></figcaption></figure>

* Укажите **«Название ресурса»** и (по желанию) выберите часовой пояс и валюту.

<figure><img src="../.gitbook/assets/ga create source ru.png" alt=""><figcaption></figcaption></figure>

* Добавьте информацию о вашей компании: выберите сферу деятельности, размер компании.

<figure><img src="../.gitbook/assets/ga add info ru.png" alt=""><figcaption></figcaption></figure>

* Укажите бизнес-цели и нажмите **«Создать»**.

<figure><img src="../.gitbook/assets/ga add goals ru.png" alt=""><figcaption></figcaption></figure>

* **Ознакомьтесь и примите условия использования** Google Аналитики.
* Добавьте новый поток данных. Для этого выберите платформу **«Веб»**.

<figure><img src="../.gitbook/assets/ga web ru.png" alt=""><figcaption></figcaption></figure>

* Введите в **«URL веб-сайта»** домен вашего сайта и напишите **«Название потока»** (например, ваш домен).

<figure><img src="../.gitbook/assets/ga add site ru.png" alt=""><figcaption></figcaption></figure>

* Закройте окно с настройкой Google Tag. В следующем окне скопируйте **«Идентификатор потока данных»**.

<figure><img src="../.gitbook/assets/ga copy id ru.png" alt=""><figcaption></figcaption></figure>

* В панели управления Wfolio перейдите в раздел настроек **«Счетчики и пиксели»**. Вставьте скопированный идентификатор потока в поле **«Google Analytics»** и сохраните изменения.

<figure><img src="../.gitbook/assets/ga add wfolio.ru.png" alt="" width="563"><figcaption></figcaption></figure>

* Готово, счетчик установлен. Теперь будет собираться статистика о посещаемости вашего сайта. Данные можно увидеть в отчетах Google Analytics.
